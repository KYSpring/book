CH2: web3.0是什么？如何产生的？

  1. 从web1.0到web3.0:

    • web1.0: 用户被动接受信息
    • web2.0: 用户通过中心化平台生产信息
    • web3.0: 用户生产信息 / 数据确权与授权 / 0信任 / 无垄断

  2. web3.0的资本推手

CH3 web3.0的核心技术——区块链

  • 区块 / 节点 / 分布式
  • 区块链三元悖论：去中心化 / 安全性 / 可扩展性
  • 共识机制：Pow / PoS / PoH / PoA / PoR / PoS
  • 预言机：区块链外信息写入区块链内的机制；
  • 区块链安全：恶意挖矿 / 双花攻击 / 供应链攻击 / 区块链拒绝服务攻击

CH4: web3.0的基本构成元素——NFT

  • 是什么？

    概念定义：Non fungiblle token 非同质化通证：一种链上数据格式。

    组成：token / 存储在链上 / 唯一编码；

    特点：不可分割、可确权、可追溯的某种信息载体；

    nft协议标准：

    • EIP 以太坊改进提案；ERC 以太坊征求意见稿；
    • 重要协议版本：ERC721/ERC1155/ERC998

  • 如何实现NFT标准协议？

    OpenZeppelin开源工具（智能合约库 / 软件开发SDK）

  • nft审美：什么样的设计才是“蓝筹”？

    • 发展初期：像素作品CRYPTOPUNK项目,

      文化层面 早期、文化符号，标记自身og身份；技术层面：ERC20

    • 价值的确定：稀缺性(一般单个项目的NFT总量是固定的) + 共识

    • 审美意识变迁：参与者的不同而不同

      女性主义主题；东方文化（On One Force）

  • NFT分类

    1. Avatar头像类

    2. 铸造NFT的机会分配：从固定时间完全公开走向白名单机制

    3. 白名单机制的底层逻辑：价值证明，从传统品牌推广走向POV模式（价值证明）【0n1 force项目】

    4. 世界观的搭建

      【Doodles】设计风格符合大众审美 / 顶级运营能力 / 社区自治 / 达成广泛的圈内外共识

      【Azuki】日系风格，拥有者才可访问其“the garden”元宇宙

    5. 音乐NFT

      • Royal平台
      • Rocki平台（基于bsc区块链）
      • Oneof平台（基于Tezos区块链）

    6. 素人自拍NFT

      • Ghozali Everyday
      • Irenezhao

    7. 体育NFT

      • NBA Top Shot 球星卡游戏

        每个球员卡都是一个NFT（普通/稀有/传奇）

    8. Sorare：足球类，基于以太坊的区块链游戏；可以组建球队参加虚拟比赛。

    9. 个人项目：梅西、奥尼尔、奥沙利文

      通常绑定明星周边：见面机会/球衣/球鞋/签名照

    10. 品牌、企业纪念之类

CH6 web3的基本组织形式——DAO

  1. DAO是什么：Decentralized Autonomous Organization

    • 去中心化： 不存在权力中心，自下而上管理；

      怎么实现去中心化

    • 自动性和自主化：智能合约实现管理的代码化/程序化和自动化
    • 可自治：决策投票
    • 公开透明：智能合约的内容公开透明
    • 高度信任、高度共识
    • token激励（可拆分token or 不可吃啊分token）

  2. DAO有何用：

    • 数字经济时代的三组关系：机器与人、平台与产品、大众与核心；
    • 平台经济下，个人价值的贡献者在平台中并未获得对等的回报；
    • dao目前的价值仍热停留在如何建立适应新时代发展需求的生产关系；【探索】
    • dao关系：组织、员工、用户三个角度；
    • “x to earn”概念：价值不被打折；生活本身变成工作；
    • 解构职业：dao给原本不被认为产生价值的行为赋予了经济价值； 打破职业化的固定发展路线 => 打破内卷；

      价值何在？也许实体价值/平台价值/个人价值是 【一种层次化的经济结构】

    • 根除垄断：生产资料共享、创意点大于用户量、不塌房；
    • 解构“用户”：web2 用户注意力 => web3用户即员工
  1. 发展史：仍然处于早期阶段
    • 2013 daniel laimer (BM) - dac；
    • 2013 v神；以太坊的创始人
    • 2016年第一个dao诞生 => 黑客通过代码漏洞转移360万个eth；
    • 2020 defi热潮 => dao实践
    • 2021 constitution DAO

  1. 各种类型的DAO：

    • cityDAO (dao能否作为一个有限责任公司存在 => 美国怀俄明州2021年4月通过地方法律承认) 筹资建立dao城市；
    • 以治理为目的的DAO: makerDao 作为治理工具
    • 为创作者而生的DAO：类似粉丝后援会
    • 投资DAO：The Lao
    • 收藏社区：
    • 社交/学习DAO： FWB / windseeker(web3咨讯) 【知识付费重构？】/ Cryto Tech Night (CTN)
    • DAO的工具：Aragon / Gnosis Safe(多账户共同管理资产) / DAOhaus

  2. 理想中的dao应该是什么样：：共同目标 + 有效决策 + 优质社区

    （再造教育？？）

    • 合规性

    • 资本暴力问题不能解决uniswap

ch7 web3.0“把经济系统嵌入互联网里”

    • web3.0的本质是生产资料的共享，但不等于资产的共享
    • 原来那些被中心化平台霸占的、本来就应该属于全体互联网用户的数据被解放了出来。可以说，普通人在web3.0中失去的只是枷锁，获得的将是整个世界。
    • 原生的支付体系，不再需要第三方支付；

  2. 一些新词/旧词新定义

    • NGMI（Not Gonna Make it, web3圈内黑话“你不行”，不属于某个共识群体）
    • MEME类似“人传人”现象，指的是由于模仿使得某种风格、思想、行为等在群体中传播：Meme是与主流文化强关联的。
    • Token： 不仅仅指的是令牌或者标记，也可以代表一种基于智能合约发出的价值流转载体；
    • 商品的新内容：NFT
    • Token vs. Coin vs. Currency区别：coin是一种只具备交易属性的链上极致流通载体；

  3. Defi去中心化金融：

    • 去中心化交易所：uniswap为例

一点不成熟的思考

借着周末休息读完了这本薄薄的web3入门书，内容并不算深入，但在一些概念上颇有触动。

web3.0在元宇宙、nft等热潮下成为了当下混沌且乏力时代的兴奋剂，让所有被迫日薄西山的行业领域为之欣喜若狂，但对于普通公众而言始终有种若即若离的疏离感。这本书算是比较好的提供了一幅相对完整的RoadMap以打破这层认知藩篱，有助于以最简单的概念逻辑结构化web3.0业已积重的碎片化概念。

“比特币最伟大的时刻就是中本聪消失的时刻。”

去中心化在web3.0中举足轻重，是其基本驱动力，由此出发衍生而出的共识机制、DApp、DAO在很大程度上都是其附庸。去中心化在3.0时代被赋予了新的语义：不再仅仅是技术专业术语而转化成了运营机制、商业模式、概念体系乃至最终构筑了web3.0的基本价值观。

“把经济系统嵌入互联网里”

智能合约实现了机器化的代管机制，而DAO则成了一种新型的生产组织形式，这势必带来地震般的社会经济结构变革。印象深刻的一段论述是：web2.0时代的用户在踏入3.0之后，其失去的只有枷锁，获得的将是无穷的价值收益。拥有绝对定价主导权的中心化平台在web3.0时代或将难以压榨任何个体价值生产者，一个从未实现过的乌托邦式的生产关系居然有可能在虚拟世界的维度被解决，这确实是让人欣喜的。但是否造就了另一种形式的乌托邦，还未可知。

“NGMI”和“Meme”模式下的信仰构筑

dao在某种程度上类似于工业革命1.0的家庭生产和2.0的集团公司组织形式，却又与之存在信仰构筑层面的极大不同，一个建构在技术信任基础上的社会群体划分正在进行中。在DID的加持下，恐怕作为第二人格的虚拟世界也在形成一套原生的“社会阶级”制度，而早期进场者极有可能获得发展时间上的先发机遇。

“人人出名五分钟”

web2.0时代的个人IP在web3.0极有可能被放大，这是一个类似战国的虚拟时代，个人英雄主义当下的web3.0中非常受欢迎。无论是以太坊还是uniswap等颠覆性应用工具，都并不需要2.0时代DAU等各类用户数据指标的限制。这是一个安迪沃霍尔预言实现得更为成功的世界。
